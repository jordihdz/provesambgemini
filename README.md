# Análisis de Ventas con Python y SQL 📊

Este proyecto forma parte de mi formación en **Analítica de Datos**. Consiste en un flujo de trabajo completo que abarca desde la creación de una base de datos local hasta la extracción y procesamiento de información mediante consultas SQL integradas en Python.

## 🚀 Tecnologías Utilizadas
* **Lenguaje:** Python 3.x
* **Librerías:** Pandas, Matplotlib, SQLite3
* **Entorno:** Jupyter Notebooks en VS Code

## 📈 Lo que aprendí en este proyecto
1. **Gestión de Entornos:** Configuración de un entorno virtual (`.venv`) para aislar dependencias.
2. **SQL Intermedio:** Uso de funciones de agregación (`AVG`, `GROUP BY`) y filtrado de datos.
3. **Integración SQL-Pandas:** Extracción de datos desde una base de datos SQLite directamente a un DataFrame.
4. **Visualización de Datos:** Generación de gráficos para interpretar resultados estadísticos.
